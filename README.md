# React Weather

#### ✨Пет-проект✨

Приложение для просмотра краткой сводки погоды

## Использованные технологии и сервисы

- React.js
- JSX
- БЭМ
- Geolocation-db API
- OpenWeather API

## Интересности

- Прелоадер
- Анимации
- Прямой доступ к местоположению (Geolocation-db)
